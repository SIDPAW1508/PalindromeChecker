# UC11: Palindrome Checker

## Description
This feature checks whether a given string is a **palindrome** using simple **two-pointer comparison**.

A palindrome is a word, phrase, or sequence that reads the same backward as forward.

## How It Works
1. Convert the input string into characters.
2. Use two pointers:
   - `start` at the beginning
   - `end` at the last character
3. Compare characters at `start` and `end`.
4. Move pointers inward until they meet.
5. If all characters match → **Palindrome**, otherwise → **Not a Palindrome**.
