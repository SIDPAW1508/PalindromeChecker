# Case-Insensitive & Space-Ignored Palindrome Checker

## Description
This feature checks whether a string is a palindrome while **ignoring spaces and letter casing**.

## How It Works
- **Normalize** the input string by removing all spaces and converting to lowercase.
- Use a two-pointer method to compare characters from the start and end of the normalized string.
- If all corresponding characters match, the string is considered a palindrome.
