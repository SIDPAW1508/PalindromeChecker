# UC8: Linked List-Based Palindrome Checker

## Description

This feature implements a **Palindrome Checker using Java LinkedList**.
The program checks whether a given string is a palindrome by comparing characters from both ends of the list.

## Logic

1. Convert the input string into characters.
2. Store each character in a `LinkedList`.
3. Compare characters from the **front and rear** using:

   * `removeFirst()`
   * `removeLast()`
4. If all characters match → **Palindrome**
5. If any pair does not match → **Not a Palindrome**

```

## Key Concept

This use case demonstrates **double-ended operations in LinkedList** for symmetric validation.
